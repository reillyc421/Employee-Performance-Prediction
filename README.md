## Employee Performance Score Prediction

This project uses classification to predict employee performance scores based on employee data. 

### Dataset
The dataset is a modified version of a [synthetic dataset](https://www.kaggle.com/datasets/desolution01/messy-employee-dataset). The file included in the repository is my modified version, which included a wider range of employee ages.

### Description
This project began with data exploration and processing using Python's pandas library. I used sckit-learn to compare several classification models before choosing one and tuning for optimal performance. Finally, I identified the most salient features used to classify the data.
