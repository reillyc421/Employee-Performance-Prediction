Predict Employee Performance Scores¶

The purpose of this project is to use a synthetic employee dataset to predict performance scores. The original dataset can be found at https://www.kaggle.com/datasets/desolution01/messy-employee-dataset. I made some modifications to make it more realistic. I will compare several classification models and optimize the best performing model.

# Evaluate

from sklearn import metrics
import seaborn as sns

def evaluate_model(model, x_test, y_test):
    y_pred = model.predict(x_test)

    acc = metrics.accuracy_score(y_test, y_pred)
    prec = metrics.precision_score(y_test, y_pred, average = 'macro')
    rec = metrics.recall_score(y_test, y_pred, average = 'macro')
    f1 = metrics.f1_score(y_test, y_pred, average = 'macro')

    confusion_matrix = metrics.confusion_matrix(y_test, y_pred)
    plt.subplots(figsize =(8,8))
    sns.heatmap(confusion_matrix, annot = True, fmt = '.0f')
    plt.xlabel('True')
    plt.ylabel('Predicted')

    result_dict = {'acc': acc, 'prec': prec, 'rec': rec, 'f1': f1, 'confusion matrix': confusion_matrix}

    return result_dict
